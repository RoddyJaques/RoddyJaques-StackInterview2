# Contents
**1 BES Data Prep.ipynb** - Prepares data from the British Election Study Post-Election Random Probability Survey to fit turnout and vote models

**2 ONS Data Prep.ipynb** - Prepares ONS Census data to be used as the Poststratification Frame 

**3 Turnout Model.ipynb** - Uses BABMI to fit and predict turnout with a Bayesian Multilevel Model

**4 VI Model.ipynb** - Uses BABMI to fit and predict vote share within each cell with a Bayesian Multilevel Model

**5 Final Poststratification Frame.ipynb** - Scales results from voting model and exports final poststratification frame


**BES-2024-General-Election-results-file-v1.0.sav** - BES 2024 general election results dataset

**Final Poststratification Frame.csv** - Final poststratification frame submission

**bes_rps_2024_1.0.1.sav** - BES Post-Election Random Probability Survey Dataset

**custom-filtered-2026-02-18T18_48_45Z.xlsx** - ONS custom census dataset
